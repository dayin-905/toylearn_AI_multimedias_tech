## 프롬프트 작성 시 주의사항 및 회고
- 프롬프트의 md형식으로 작성이 계속해서 제대로 먹지 않아 추가 요청 작업을 하였으나 꾸준하게 누락하거나 스스로 변경함.
- md형식 변경은 gemini는 거의 작동하고 있지 않기 때문에 우선적으로 본인이 추가 작업함.
- 어제 피드백을 바탕으로 모든 것을 AI에게 모든 것을 물어보고 진행해도 된다는 팁을 깨달은 뒤 모든 작성을 AI에게 맡김.
- 내가 작성한 프롬프트는 실제로 얼마 되지 않는다.
- 다음은 퍼블릭시티 같은 AI 어시스턴트를 활용해볼 것.
```
해당 문제 상황에 대해서 페르소나, 배경, 핵심 문제, 해결 과제를 보고 너 스스로 [해결 과제]를 풀어봐.
내가 주는 자료 [AI 프롬프트 전략]을 활용해서 [작성 내용]을 작성해주면 돼.

[해결 과제]
1. 가상 문제 상황
스크래핑

2. 훈련 초점(힌트)
스크래핑

[작성 내용]
1. 프롬프트 유형과 선택 이유
2. 작성을 위해 추가 필요 요건(부족한 자료나 문제 제공 등)
3. 문제 해결 프롬프트
4. 프롬프트 실행 결과

[AI 프롬프트 전략]
```

## 1) 프롬프트 유형과 선택 이유

- **조합 유형**: 역할 부여(역할: 시니어 멀티미디어 콘텐츠 매니저) + 지시형(구체 작업 지시) + 제약 조건(길이·포맷·톤) + 재구성/번역(롱폼→숏폼/텍스트 변환) + 예시 기반(샘플 출력 형식 제공).
- **이유**: 다양한 산출물(타임스탬프, 숏클립 시나리오, 캡션, 블로그 글)이 필요하므로 역할 부여로 전문성 확보하고, 제약 조건으로 플랫폼별 규격을 강제하며 재구성·예시를 섞어 출력 일관성을 보장한다.

- **보조 유형**: CoT(단계별 작업 플로우) — 편집 파이프라인(컷 분할→자막→색보정→포맷 변환)을 단계별로 도출하기 위해 사용.

## 2) 작성을 위해 추가 필요 요건 (부족한 자료)

현재 원본 영상 + 스크립트만 사용 가능하므로 반드시 필요하거나 있으면 좋은 자료:

1. 브랜드 로고(벡터/PNG), 브랜드 색상/폰트 가이드 — 썸네일·자막 일관성 위해 필수 권장.

2. 제품 고해상도 스틸(없다면 영상 프레임에서 추출) — 블로그 썸네일·캐러셀용.

3. 사용 가능한 음악/라이선스 정보(유료·오픈) — 플랫폼별 애드머니·저작권 관리.

4. 타깃 KPI(예: 유튜브 조회수/구독 목표, 인게이지먼트 기준, CPA 등) — 우선순위 및 CTA 설계에 필요.

5. (선택) 고객 승인 가능한 문구/금지 문구(예: '무첨가' 표현 금지 등) — 법적·브랜드 안전 보장.

만약 위 자료 중 일부가 없다면 대체안(예: 영상에서 추출한 스틸 사용, 로열티 프리 음악 제안, 일반적 브랜드 톤 적용)을 적용한다 — 아래 기획안에 이미 대체안 포함.

## 3) 문제 해결 프롬프트 (AI에게 실제로 넣을 프롬프트 — 복수, 플랫폼별)

아래 프롬프트들은 그대로 편집팀의 AI/에디터에게 붙여넣어 실행 가능하도록 구성했습니다. (한 번에 여러 프롬프트를 연속 실행하면 효율적임)

### A. 전체 전략·클립 분할 생성 (역할+CoT+제약)
```
당신은 10년차 멀티미디어 콘텐츠 매니저입니다. 제공된 15분 고화질 홍보 영상(링크/로컬 파일)과 스크립트를 분석해, 다음을 단계별로 만들어 주세요.

1) 15분 원본을 유튜브(롱폼), 인스타 릴스(<=60s), 틱톡(<=60s), 인스타 캐러셀(이미지+텍스트), 네이버 블로그(텍스트 기반)용으로 재가공하기 위한 '컷 리스트'와 '타임스탬프'를 생성하라. 각 출력을 위해:
 - 유튜브: 4~6개 챕터(타임스탬프 포함), 각 챕터별 10~30초 요약(유튜브 설명에 넣을 문장)
 - 숏폼(인스타/틱톡): 훅 3초, 핵심 메시지 15~40초, CTA 3~5초 형태로 **최소 6개** 숏클립 후보(각 후보마다 시작·끝 타임코드, 핵심 자막 문구, 세로크롭 프레임 제안)
 - 캐러셀: 6슬라이드 구성(각 슬라이드에 사용할 권장 프레임 타임코드 + 1문장 캡션)
 - 네이버 블로그: SEO 최적화 제목 1개, 메타디스크립션, 본문(약 450~650자), 삽입권장 이미지 타임스탬프 3장

2) 각 플랫폼별로 **포맷 규격(해상도, 길이), 권장 자막 스타일(폰트 크기/길이 제한), CTA 문구 3가지**를 제시하라.

제약: 추가 촬영 불가, 원본과 스크립트만 사용. 브랜드 색상/로고가 없으면 영상에서 추출한 색상을 사용하고, 음악은 로열티 프리로 제안. 출력은 표와 리스트 형식으로 제공.
```

### B. 숏폼 개별 편집 프롬프트 (예시 기반 + 제약)
```
당신은 경험 많은 숏폼 영상 편집자입니다. 아래은 클립 후보 #1 정보입니다:
- 원본 시작 02:10, 끝 03:00
- 스크립트 요약: "신선한 재료가 바로 집으로 — 10분 조리로 완성되는 간편함"
이 클립을 인스타 릴스(<=60s)와 틱톡(<=60s)용으로 최적화해 편집 지시문을 만들어 주세요. 포함 항목:
- 세로 자르기(프레임 기준) 및 중요 화면 영역(x축,y축 비율)
- 훅(첫 3초)용 텍스트 및 애니메이션(예: 큰 볼드 텍스트 슬라이드인)
- 15~30초 스토리보드(컷 목록, 필요한 자막 문구)
- 배경음악 유형(템포/무드), 볼륨 믹싱(음성 대비)
- 권장 캡션(문장 1) 및 해시태그 5개
제약: 편집시간 30분 이내로 제작 가능하도록 컷 최소화.
``` 

### C. 네이버 블로그(텍스트 변환) 프롬프트 (재구성/제약)
```
당신은 SEO에 능한 콘텐츠 라이터입니다. 제공된 스크립트와 유튜브 챕터 요약을 바탕으로 네이버 블로그 게시글을 작성해 주세요.
요구사항:
- 제목: '친환경 밀키트' 관련 핵심 키워드 포함(예시: '친환경 밀키트, 신선함, 간편 조리')
- 메타디스크립션 120자 내
- 본문 500~700자, 소제목 2개 이상 포함, 제품 장점(신선함·간편성·친환경) 각 1문단씩
- 본문 내에 CTA(제품 페이지 링크/구매 유도) 1개 포함
- 삽입 이미지 권장(타임스탬프 3개)과 이미지 alt 텍스트
제약: 스크립트 문장을 그대로 복사하지 말고 자연스럽게 재작성하라.
```

### D. 자동화용 메타데이터/제목·설명 생성 프롬프트 (조건부)
```
유튜브 업로드용 메타데이터 생성:
- 입력: 영상 스크립트 요약 + 4개 챕터 요약
출력: SEO 최적화 제목(수정안 3개), 상세 설명(첫 문단 150자 내, 전체 400~600자), 15개 태그(콤마 구분), 3개 썸네일 텍스트(한 줄), 권장 썸네일 이미지(타임스탬프)
제약: 클릭베이트 과장 금지, 제품의 핵심(신선함·간편성·친환경) 반드시 포함.
```

## 4) 프롬프트 실행 결과 — (실제 기획안 출력)

아래는 위 프롬프트를 실행했을 때 즉시 쓸 수 있는 **구체적 산출물(편집 지시서 + 게시물 템플릿)** 입니다. (편집팀에게 바로 전달 가능)

### A. 전체 워크플로우 요약 (한눈에)

**1. 분석(10분)**: 스크립트 자동 요약 → 핵심 메시지 6개 추출

**2. 컷 분할(30분)**: 타임라인에 서브클립(유튜브 챕터·숏폼 후보·캐러셀 스틸) 생성

**3. 배치 편집**(각 플랫폼별 프리셋 사용):

- 유튜브: 16:9, 4K->Color grade light, B-roll 삽입, 챕터 마커

- 숏폼: 9:16 프레임, 자동 캡션, 빠른 컷(1~3s) 사용, 훅 텍스트 애니메이션

- 캐러셀: 2:3(4:5) 비율 스틸 추출 + 텍스트 레이아웃

- 블로그: 스크립트 재작성 + 이미지(프레임) 삽입

**4. 자막·자막 번역(자동 후 수동 교정)**

**5. 내보내기·업로드 메타데이터 삽입** (각 플랫폼 규격 파일명 규칙 적용)

### B. 유튜브(롱폼) — 산출물

**권장 제목(3안)**

1. [브랜드]의 친환경 밀키트 — 신선함 그대로, 집에서 10분 완성 레시피

2. 신선한 재료 × 간편 조리: [브랜드] 친환경 밀키트 실사용 리얼 리뷰(조리·보관 팁)

3. 친환경 밀키트 제조부터 조리까지: 재료·포장·맛을 그대로 담은 15분 하이라이트

**- 챕터(권장, 타임스탬프 예시)**
00:00 인트로 — 브랜드 & 제품 핵심 메시지 (요약문: “친환경 재료로 편리한 식사”)

01:10 재료 출처 & 신선도 강조 (요약 문장)

04:20 제조 과정(포장/보관) 하이라이트

07:10 조리 데모(10분 조리) — 단계별 포인트

11:00 맛·플레이팅 및 서빙 팁

13:30 친환경 포장·지속가능성 이야기 + 클로징 CTA

**- 유튜브 설명(첫문단 150자 예)**
“[브랜드]의 친환경 밀키트를 15분짜리 영상으로 담았습니다. 신선한 재료 출처, 간편한 조리법, 그리고 지속가능한 포장까지—영상에서 바로 확인하세요. ⬇️ 제품/구매 링크”

**- 태그(예)**: 친환경밀키트, 밀키트추천, 간편요리, 지속가능포장, 신선식품, 집밥, 10분요리, [브랜드], 친환경

**- 썸네일 텍스트(3안)**

- “10분으로 완성되는 신선함”

- “친환경 밀키트 실사용”

- “포장부터 조리까지, 모든 것”

**- 편집 노트**

- 오프닝 5초에 제품 로고(비어있으면 영상 프레임에서 추출) + 강한 훅(“10분이면 완성!”)

- B-roll: 재료 클로즈업(1.5x 속도 슬로우 모션 섞기)

- 자막은 한 줄 / 48자 내 — 브랜드 톤(격식+따뜻함) 유지

- 색보정: 자연스러운 색온도 + 채도 소폭 상향(식품 맛 표현)

### C. 인스타그램 릴스 / 틱톡 — 숏폼 6개 후보 (각 후보에는 타임코드·훅·자막 포함)

포맷 공통 가이드: 세로 9:16, 최대 60초(권장 15~30s), 첫 3초 훅 강력, 자동 캡션 필수, 영상 중앙 인물·제품 배치 유지

#### 숏폼 후보 1 — “훅: 10분 조리”

- 타임스탬프: 07:05–07:40 (조리 빠른 요약)

- 구성:

    - 00–03s 훅 텍스트: “10분이면 집밥 완성🍽️” (큰 볼드 애니)

    - 03–20s 핵심: 조리 핵심 컷(재료→팬→완성) / 자막: “한 번에 준비→10분 조리”

    - 20–25s CTA: “더 많은 레시피는 프로필 링크”

- 해시태그: #10분요리 #밀키트 #간편요리 #친환경

- 음악: 빠른 템포의 밝은 팝(로열티 프리) — 보이스 인게이지먼트 우선

### 숏폼 후보 2 — “훅: 신선함 클로즈업”

- 타임: 01:50–02:30 (재료 클로즈업)

- 구성: 훅(재료 슬로우모션) → '생생함' 강조 자막 → 맛 묘사 짧은 인터컷

- 캡션 예: “포장만 풀면 신선함 그대로”

#### 숏폼 후보 3 — “훅: 친환경 포장”

- 타임: 11:40–12:10 (포장·분해 샷)

- 구성: 포장 소개→포장 재료 강조(텍스트: '재활용 가능')→CTA

#### 숏폼 후보 4 — “10초 레시피 팁(플레이팅)”

- 타임: 09:30–09:55 (플레이팅 팁)

- 포맷: BGM 약하게, 텍스트 팁 2개 삽입

#### 숏폼 후보 5 — “사용자 리액션 컷(맛 표현)”

- 타임: 12:30–13:00 (테이스트 리액션)

- 용도: 리뷰형 숏클립, 유튜브 하이라이트 트레일러

#### 숏폼 후보 6 — “비포&애프터(조리 전/후)”

- 타임: 06:40–07:20 (재료 vs 완성 샷)

- 구성: 스플릿 화면 → 전환 매직 컷 → CTA

### D. 인스타그램 캐러셀 — 6 슬라이드 구성 (이미지: 영상 프레임 추출 권장)

**본문(약 520자, 소제목 포함)**

신선함을 집으로 — [브랜드] 친환경 밀키트 리뷰
요즘 밀키트는 ‘간편함’뿐 아니라 ‘신선함’과 ‘지속가능성’이 중요하다. [브랜드]의 친환경 밀키트는 지역 농가에서 당일 수확한 재료를 사용해 포장 직전까지 신선함을 유지한다는 점이 인상적이었다. 영상 속 조리 장면을 통해 10분 안에 완성되는 간단한 조리법도 확인할 수 있었다.

**조리: 10분이면 충분하다**

핵심은 준비 단계의 효율성이다. 모든 재료가 계량·포장되어 있어 별도 손질이 거의 필요 없다. 영상 07:10 구간을 보면 재료를 바로 팬에 넣고 10분 내 완성하는 과정을 확인할 수 있다.

**친환경 포장과 보관 팁**
포장은 재활용 가능한 소재를 사용해 환경 부담을 낮췄다(11:40 구간 참조). 보관은 냉장 기준으로 3일 권장, 장기간 보관 시 별도 냉동 권장 등 실용적 안내도 포함되어 있다.

제품 구매는 [제품 링크]에서 확인 가능하며, 첫 구매 시 사용할 수 있는 할인 코드(예: FIRST10)를 제공한다. 영상 하단 설명란에서 더 많은 조리 팁과 챕터를 확인하세요.
