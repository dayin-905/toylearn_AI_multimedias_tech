## 🧠 4️⃣ 역량 분석 및 교육 필요도 (Competency Analysis)
- **역량평가 항목별 점수 분석**
  - 직무 이해도 / 문제 해결력 / 협업 능력 / 리더십 / 창의성  
- **부서별 요구사항 도출**
  - 콘텐츠팀: 크리에이티브 사고력 강화  
  - 전략기획팀: 데이터 기반 분석 교육  
  - 제작팀: 최신 영상 편집·촬영 기술 습득  
  - 운영팀: 프로세스 효율화 및 리스크 관리  
- **우선순위 기준**
  - 전략 중요도 + 실무 필요도 + 직원 요청 빈도
